# Meety

Cherish and commemorate your IRL meeting by minting an NFT, creating a personalized and verifiable memory that lasts forever!

This project has two main folders:

```bash
.
├── contracts
│   ├── src
│   ├── broadcast
│   ├── script
│   └── test
└── web
    ├── app
    └── src
```

[contracts](/contracts/README.md) folder contains all the solidity code

[web](/web/README.md) folder contains all the frontend code
